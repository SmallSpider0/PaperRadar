# PaperRadar Seed Data

This directory contains a compressed PostgreSQL seed dump with the paper metadata and derived processing results included with the public repository.

- File: `paperradar-paperdata.sql.gz`
- Compressed size: `58M`
- SHA-256: see `paperradar-paperdata.sql.gz.sha256`

The seed contains paper metadata, metadata embeddings, topic profiles, topic-profile run logs, and query cache/rule tables. It does not include private user sessions, passwords, API keys, or local secret files.
