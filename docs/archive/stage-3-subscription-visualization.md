# 阶段三：订阅完善与可视化扩展

增强持续跟踪能力与信息呈现能力，让 PaperRadar 成为稳定的研究跟踪台。

## 功能范围

- 组合检索订阅
- 作者订阅
- 更丰富的过滤器
- Webhook / 邮件推送
- 趋势图与统计面板
- 作者 / 主题关系图
- 配图生成工作流（可选）

## 核心交付物

- advanced subscriptions
- notification channels
- analytics dashboard
- visualization modules
- figure generation workflow（optional）
